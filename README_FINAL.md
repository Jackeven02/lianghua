## 📈 使用示例

### GUI操作流程

1. **启动程序**
   ```bash
   python start_gui.py
   ```

2. **设置参数**
   - 扫描数量: 10只
   - 最低评分: 40分

3. **开始扫描**
   - 点击"开始扫描市场"按钮
   - 等待扫描完成（约10-20秒）

4. **查看结果**
   - 表格显示所有符合条件的股票
   - 点击任意行查看详细信息

5. **分析决策**
   - 关注买入信号的股票
   - 查看目标价和止损价
   - 阅读投资理由

## 🆘 常见问题

### Q: GUI窗口打不开？
A: 检查PyQt5是否安装: `pip list | findstr PyQt5`

### Q: 数据获取失败？
A: 检查网络连接，确认efinance服务可用

### Q: 扫描速度慢？
A: 正常现象，每只股票需要1-2秒，减少扫描数量可加快速度

### Q: 没有找到投资机会？
A: 降低最低评分阈值，或增加扫描数量
