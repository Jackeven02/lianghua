## 📋 可用程序

### 1. 启动菜单 (推荐)
```bash
python start.py
```
提供友好的菜单界面，可以选择不同功能。

### 2. 快速测试
```bash
python quick_start.py
```
快速验证efinance数据连接是否正常。

### 3. 单股分析
```bash
python demo_advisor.py
```
分析单只股票（贵州茅台），展示完整的分析过程和投资建议。

### 4. 市场扫描
```bash
python demo_scanner.py
```
扫描10只热门股票，找出最佳投资机会。

## 📊 功能特点

✅ **真实数据** - 使用efinance获取真实A股数据  
✅ **技术分析** - 计算多种技术指标（均线、RSI等）  
✅ **智能评分** - 多维度评分系统  
✅ **投资建议** - 生成具体的买入/持有/观望信号  
✅ **价格目标** - 自动计算目标价和止损价
