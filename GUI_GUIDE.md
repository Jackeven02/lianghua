## 🎯 使用步骤

### 1. 设置扫描参数

**扫描数量**
- 范围: 5-20只股票
- 默认: 10只
- 说明: 扫描的股票数量，数量越多耗时越长

**最低评分**
- 范围: 0-100分
- 默认: 40分
- 说明: 只显示评分高于此值的股票

### 2. 开始扫描

点击 **"🔍 开始扫描市场"** 按钮

- 进度条会显示扫描进度
- 状态栏显示当前分析的股票
- 扫描期间按钮会被禁用

### 3. 查看结果

**结果表格**
- 按评分从高到低排序
- 绿色背景表示买入信号
- 点击任意行查看详细信息

**表格列说明**
- **代码**: 股票代码
- **名称**: 股票名称
- **信号**: 投资建议（强烈买入/买入/持有/观望）
- **评分**: 综合评分（0-100）
- **信心度**: 建议的可信度
- **当前价**: 最新价格
- **目标价**: 建议目标价（+15%）
- **止损价**: 建议止损价（-8%）
- **RSI**: 相对强弱指标

### 4. 查看详情

点击表格中的任意行，下方会显示：
- 完整的价格信息
- 技术指标详情
- 投资理由分析

## 📈 功能说明

### 分析指标

**技术面分析**
- 移动平均线（SMA 5/20/60）
- 相对强弱指标（RSI）
- 成交量分析
- 价格动量

**评分系统**
- 均线排列: 30分
- RSI指标: 20分
- 价格位置: 15分
- 成交量: 10分
- 短期动量: 10分

**信号等级**
- 60分以上: 强烈买入/买入
- 40-59分: 持有
- 40分以下: 观望

### 价格建议

**目标价**
- 计算方式: 当前价 × 1.15
- 预期收益: +15%

**止损价**
- 计算方式: 当前价 × 0.92
- 风险控制: -8%

## 🐛 常见问题

### Q1: 窗口无法打开
**解决方案:**
```bash
# 检查PyQt5是否安装
pip list | findstr PyQt5

# 重新安装
pip install PyQt5
```

### Q2: 扫描速度慢
**原因:** 需要从网络获取数据

**解决方案:**
- 减少扫描数量
- 确保网络连接良好
- 耐心等待（约1-2秒/股）

### Q3: 没有找到投资机会
**原因:** 所有股票评分都低于最低评分

**解决方案:**
- 降低最低评分阈值
- 增加扫描数量
- 更换股票列表

### Q4: 数据获取失败
**原因:** 网络问题或股票代码错误

**解决方案:**
- 检查网络连接
- 验证股票代码是否正确
- 稍后重试

## 💡 使用技巧

### 1. 快速筛选
- 设置较高的最低评分（如60分）
- 只关注买入信号的股票

### 2. 全面分析
- 设置较低的最低评分（如30分）
- 查看所有股票的表现

### 3. 定期扫描
- 每日收盘后扫描一次
- 关注评分变化趋势
- 及时调整持仓

### 4. 结合其他分析
- 查看公司基本面
- 关注行业动态
- 参考市场情绪
